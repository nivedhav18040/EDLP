Frequently Asked Questions

Answers to common evaluation questions

Can I get a demo of Endpoint DLP Plus?

Yes, you can request a personalized demo to see how discovery, audit, and enforcement work in your environment.

What detection methods are available in built-in templates?

Endpoint DLP Plus supports four built-in rule types: Regex, Fingerprinting, Keyword, and File Extension based detection.

What happens if multiple policies apply to the same action?

Policies are merged and applied in precedence order: Block (highest), Allow within trusted (lower), and Audit (lowest). If any policy blocks an action (for example, file upload), it overrides allow/audit policies for that action.

Can I start in audit mode before blocking anything?

Yes. You can begin with audit-only monitoring to understand sensitive file movement, then switch to enforcement once your policies are validated.
