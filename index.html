Frequently Asked Questions

Answers to common evaluation questions

Can I get a demo of ManageEngine Endpoint DLP Plus?

Yes. You can request a personalized demo to see how discovery, auditing, and enforcement will work in your environment.

What rule types are available for identifying sensitive data?

Endpoint DLP Plus supports four built-in rule types: RegEx, Fingerprinting, Keyword, and File extension-based rules. You can use these as-is or tailor them to your needs.

How do multiple policies work together?

All applicable policies are merged and applied in a clear precedence order: Block (highest) → Allow within trusted (lower) → Audit (lowest). So even if just one policy blocks an action (like file upload), it overrides any other policy that audits or allows within trusted.
