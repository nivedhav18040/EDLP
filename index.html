Data discovery finds sensitive data, and data controls stop it from leaving your endpoints.

Data discovery & classification

  • AI-based detection Automatically identifies sensitive content patterns to reduce manual effort.
  • Rule-based detection (RegEx, keyword, file extension) Use simple rules for quick coverage across common file types and formats.
  • Context-based (source-aware) classification If a file comes from a known business source (for example, a CRM download), it can be tagged as sensitive right away.

Data transfer, audit & control

  • Audit sensitive file movement See where sensitive files are going before you enforce rules.
  • Limit which apps can open sensitive files Prevent sensitive files from being opened by untrusted desktop apps.
  • Prevent uploads to web apps Stop sensitive files from being uploaded to browsers and cloud services.
  • Printing restrictions Control printing of sensitive files and reduce accidental sharing.
  • Control USB, screen capture/recording, and clipboard Block copying sensitive content to USB drives, screenshots/recordings, and clipboard paste actions.
