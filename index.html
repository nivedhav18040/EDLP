FAQ

Frequently asked questions

Short, practical questions for quick clarity.

What are data rules, and what do they do? +
Data rules define how Endpoint DLP detects sensitive data. Inbuilt template data rules support four types: RegEx, Fingerprinting, Keyword, and File Extension based rules.
When should I use inbuilt rule types vs custom rules? +
Use inbuilt rule types when you need faster setup for common detection patterns. Use custom rules when you need organization-specific patterns, formats, or logic tailored to your workflows.
What happens if multiple policies apply to the same endpoint? +
Policies are merged and applied by precedence:

Block (Highest) ‚Üí Allow within trusted (Lower) ‚Üí Audit (Lowest)

Example: If any one policy has Block for a control like File Upload, it will override other policies that are set to Audit or Allow within trusted for the same control.

Quick rollout tip

Start in Audit mode to observe real behavior. Then move to Enforce mode for high-risk actions like cloud uploads, external email attachments, and printing workflows.
